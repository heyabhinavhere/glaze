# Glaze — Parameter Model

## Overview

The parameter model is the brain of the tool. It defines every adjustable property of the glass effect, how properties relate to each other, and how they translate to CSS output.

The model has two layers:
- **Simple Mode**: 5 smart controls that each move multiple CSS layers in coordinated ways
- **Advanced Mode**: Individual override controls for each CSS layer

## Design Principle: Figma/Apple Language, Not CSS Language

Users think in terms of "Light direction" and "Depth," not "box-shadow offset" and "border gradient angle." The tool's controls use Figma/Apple terminology. The CSS translation happens internally.

Reference: Figma's glass effect panel uses these controls:
- Light (angle + intensity)
- Refraction
- Depth
- Dispersion
- Frost
- Splay

---

## Simple Mode Controls

### 1. Light
Controls where the "light source" is relative to the glass panel. This single control coordinates three CSS layers: border gradient direction, inner shadow direction, and specular highlight position.

- **Angle**: 0° to 360°, default: 315° (top-left, matches Figma's -45°)
  - Visual control: a circular angle picker (not a slider)
  - 0° = light from the right, 90° = light from bottom, 180° = light from left, 270° = light from top, 315° = light from top-left
- **Intensity**: 0% to 100%, default: 80%
  - Controls how bright the lit border edge is and how dim the opposite edge is
  - At 0%: uniform dim border all around
  - At 100%: strong contrast between lit and unlit edges

### 2. Depth
Controls how "thick" the glass feels — how much it separates from the background.

- **Range**: 0 to 100, default: 42 (matches Figma reference)
- **CSS layers affected**:
  - Inner shadow: blur scales from 0px to 8px, opacity from 0% to 20%
  - Drop shadow: Y offset scales from 0px to 16px, blur from 0px to 48px, opacity from 0% to 25%
  - Border glow spread: softness scales from 0px to 4px

### 3. Blur
Controls how much the background is blurred/frosted behind the glass.

- **Range**: 0 to 100, default: 20
- **CSS layers affected**:
  - `backdrop-filter: blur()`: scales from 0px to 60px
  - `backdrop-filter: saturate()`: scales from 1.0 to 1.8 (higher blur = more saturation to prevent washed-out look)
- At 0: fully transparent, you see the background clearly
- At 100: heavily frosted, background is a soft color wash

### 4. Tint
Controls the color wash over the glass surface.

- **Color**: hex color picker, default: #FFFFFF
- **Opacity**: 0% to 50%, default: 12% (matches Figma reference: FFFFFF at 12%)
- **CSS output**: `background: rgba(R, G, B, opacity)`
- Smart defaults:
  - When background is light: suggest white tint at 12%
  - When background is dark: suggest white tint at 8% or black tint at 20%

### 5. Grain
Controls the noise/texture overlay that makes the glass feel physically real.

- **Enabled**: toggle, default: on
- **Intensity**: 0% to 10%, default: 3%
- **CSS implementation**: SVG `<feTurbulence>` filter applied as a pseudo-element overlay
- Maps to Figma's "Frost" parameter (Figma reference: Frost: 8)

## Advanced Mode Controls

These allow direct override of individual CSS layers. When a user changes an advanced control, the corresponding simple control shows a "custom" indicator.

### Border
- Light-side opacity: 0% to 100%, default: 40%
- Dark-side opacity: 0% to 20%, default: 5%
- Width: 0.5px to 3px, default: 1px
- Softness (outer glow): 0 to 20px, default: 2px
- Note: border is implemented as a gradient that follows the Light angle. Bright on the light-facing edges, dim on the opposite.

### Inner Shadow
- Follows light angle: toggle, default: on
- Manual angle: 0° to 360° (only when "follows light" is off)
- Blur: 0 to 20px, default: 4px
- Spread: 0 to 10px, default: 0px
- Opacity: 0% to 30%, default: 10%
- Color: default: #000000

### Drop Shadow
- X offset: -20 to 20px, default: 0
- Y offset: 0 to 40px, default: 8px
- Blur: 0 to 60px, default: 24px
- Spread: -10 to 10px, default: 0px
- Opacity: 0% to 40%, default: 12%
- Color: default: #000000

### Specular Highlight
- Enabled: toggle, default: off (this is subtle and optional)
- Follows light angle: toggle, default: on
- Size: 10% to 50% of element, default: 25%
- Intensity: 0% to 30%, default: 10%
- Implementation: radial gradient positioned near the light source corner

### Backdrop
- Blur: 0 to 100px, default: 20px
- Saturation: 0.5 to 2.0, default: 1.2
- Brightness: 0.8 to 1.3, default: 1.0

## Presets

Each preset is a complete config object. These must be hand-tuned by visually inspecting the result, not generated by guessing values. The builder should tune these in the browser until they look genuinely premium.

### Apple Thin
- Very transparent, low blur (8px), subtle border, minimal depth
- Use case: overlays where you want content visibility

### Apple Regular
- The balanced default. Matches the Figma reference values.
- Light: 315° at 80%, Depth: 42, Blur: 20, Tint: #FFFFFF at 12%, Grain: 3%

### Apple Thick
- Higher blur (30px), stronger tint (20%), deeper shadows, heavier depth
- Use case: modal panels, cards with text that needs readability

### visionOS Panel
- Wider border glow, specular highlight enabled, more pronounced depth
- Use case: floating panels, spatial UI elements

### Subtle Dark
- Dark tint (#000000 at 20%), low opacity border, muted everything
- Use case: dark mode UI, dark backgrounds

### Frosted
- Heavy blur (40px), strong grain (5%), muted tint
- Use case: classic frosted glass look, photo overlays

## Config Object Shape

```typescript
interface GlassConfig {
  // Simple mode values
  light: {
    angle: number;      // 0-360, default 315
    intensity: number;  // 0-100, default 80
  };
  depth: number;        // 0-100, default 42
  blur: number;         // 0-100, default 20
  tint: {
    color: string;      // hex, default "#ffffff"
    opacity: number;    // 0-50, default 12
  };
  grain: {
    enabled: boolean;   // default true
    intensity: number;  // 0-10, default 3
  };

  // Advanced overrides (null means "auto-derive from simple mode")
  overrides: {
    border: {
      lightOpacity: number | null;   // 0-100
      darkOpacity: number | null;    // 0-20
      width: number | null;          // 0.5-3
      softness: number | null;       // 0-20
    } | null;
    innerShadow: {
      followsLight: boolean;
      angle: number | null;
      blur: number | null;
      spread: number | null;
      opacity: number | null;
      color: string | null;
    } | null;
    dropShadow: {
      x: number | null;
      y: number | null;
      blur: number | null;
      spread: number | null;
      opacity: number | null;
      color: string | null;
    } | null;
    specular: {
      enabled: boolean;
      followsLight: boolean;
      size: number | null;
      intensity: number | null;
    } | null;
    backdrop: {
      blur: number | null;
      saturation: number | null;
      brightness: number | null;
    } | null;
  };

  // Meta
  preset: string | null;   // name of preset if one is active, null if custom
  borderRadius: number;     // px, default 20
}
```

## CSS Translation Logic

The core engine is a pure function: `glassConfigToCSS(config: GlassConfig) => CSSOutput`

This function takes the config object and produces:
1. The main element styles (background, border-radius, position: relative, isolation: isolate)
2. backdrop-filter styles
3. Border styles (using a gradient border technique — either border-image or a pseudo-element with mask)
4. box-shadow (combining inner shadow and drop shadow)
5. The grain overlay (SVG filter as pseudo-element ::after)
6. Optional specular highlight (radial gradient as pseudo-element ::before)

The function should also output variants for:
- Vanilla CSS (class-based)
- Tailwind (arbitrary values)
- React inline styles + component wrapper
- SwiftUI modifiers
- Natural language AI prompt
- Raw JSON config
